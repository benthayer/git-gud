This level is designed to help you merge.
>>>
Make a new branch called bugFix by typing the command "git branch bugFix"
>>> 
Checkout the bugFix branch with "git checkout bugFix"
>>>
Commit once ("git gud commit")
>>>
Go back to master with "git checkout master"
>>>
Commit another time the same way that you did before
>>>
Merge the branch bugFix into master with "git merge bugFix". The command "git log --graph --oneline" is a useful tool to observe the newly merged tree.
